Posted on 2019-09-08 Edited on 2021-05-11 In 环境配置
在 IntelliJ IDEA 中使用 Antlr4，根据输入自动生成语法树并可视化

ANTLR (ANother Tool for Language Recognition)

步骤

1.安装插件
ANTLR v4 grammar plugin

2.新建一个Maven项目，在pom.xml文件中添加ANTLR4插件和运行库的依赖。

查找maven依赖
搜索：org.antlr.v4.runtime maven repository

在pom.xml里加入以下依赖

注意

Maven要用默认的

3.在src/main/java/antrl中导入 Antlr 的文法文件Java8.g4。

4.右键Java8.g4，选择Configure ANTLR，配置output路径。

5.右键Java8.g4，选择Generate ANTLR Recognizer。

Lexer是Antlr生成的词法分析器，Parser是Antlr生成的语法分析器。
